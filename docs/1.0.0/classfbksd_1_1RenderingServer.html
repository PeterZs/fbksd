Detailed Description

Implements the server that provides samples and scene information to FBKSD.

this class uses a callback mechanism. The renderer should provide the appropriate callback functions that will be called when the client makes the corresponding request.

◆ onEvaluateSamples()

void RenderingServer::onEvaluateSamples ( const EvaluateSamples &  callback)

Sets the EvaluateSamples callback.

The callback is called every time the client asks for samples to be rendered. The parameters passed to the callback are:

  • spp: number of requested samples per pixel
  • remainingCount: an extra number of samples (not multiple of the number of pixels)

The callback should return true on success.

Callback signature:

bool callback(int64_t spp, int64_t remainingCount);
