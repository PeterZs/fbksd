Converts a string to a Feature. Returns true if succeeded.

The string should not contain a number suffix. To obtain the numbered version of a feature, use toNumbered(). Ex: To obtain WORLD_X_1, use

Feature f;
stringToFeature("WORLD_X", &f);
f = toNumbered(f, 1);
