Detailed Description

Helper class for accessing samples in a tile.

A BufferTile is basically a wrapper for a tile of memory used to transfer sample data between client and server.

When using the SPP variants of the BenchmarkClient sample evaluation methods, this wrapper is used to provide a more convenient way of accessing samples using (x,y) pixel positions.
